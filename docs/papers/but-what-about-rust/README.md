# But What About Rust?

Design-study paper on building a Rust-compatible implementation from the current Java `rmatch` codebase using Codex-assisted, scaffolded migration.

## Files

- `but-what-about-rust.tex` - LaTeX source
- `but-what-about-rust.pdf` - rendered PDF
- `references.bib` - bibliography
- `Makefile` - build targets

## Build

```bash
make pdf
```

## Focus

- Defines required scaffolding artifacts for reliable migration
- Proposes translation-validation and differential-testing workflow
- Anchors the plan in relevant scientific and engineering literature
